# Doenca_cardiacaML

Este projeto tem como objetivo desenvolver um modelo de Machine Learning capaz de prever a probabilidade de um indivíduo desenvolver doenças cardíacas com base em dados clínicos e fatores de risco. A proposta busca auxiliar profissionais de saúde na identificação precoce de pacientes com maior risco, contribuindo para diagnósticos mais rápidos e estratégias de prevenção mais eficazes.
